## 🔗 Live Demo

👉 https://dinesha94.github.io/Coffee-Shop-Website/

## 📌 Overview

This project is a multi-section responsive website built to represent a modern coffee brand. It focuses on:
 - Clean and aesthetic layout
 - Smooth user experience across all devices
 - Structured and maintainable front-end code
The website includes key sections typically found in real business websites such as Hero, About, Menu, Testimonials, Gallery, and Contact.

## ✨ Key Features

#### 🎨 UI/UX Design
- Modern and minimal coffee-themed interface
- Consistent typography and color palette
- Smooth scrolling and section transitions

####  📱 Responsive Design
- Mobile-first approach
- Optimized for:
  - 📱 Mobile devices
  - 💻 Laptops
  - 🖥️ Large screens
    
### ⚡ Interactivity
  - Navigation menu toggle (mobile view)
  - Smooth scrolling between sections
  - Dynamic UI behavior using JavaScript

### 🧩 Sections Included
  - Hero – Engaging landing section with CTA
  - About – Brand story and introduction
  - Menu – Coffee items display
  - Testimonials – Customer feedback
  - Gallery – Visual showcase
  - Contact – User inquiry form
  - Footer – Social links & quick navigation

## 🛠️ Tech Stack
| Technology            | Purpose                            |
| --------------------- | ---------------------------------- |
| HTML5                 | Structure and semantic layout      |
| CSS3 (Flexbox & Grid) | Styling and responsive layout      |
| JavaScript (ES6)      | Interactivity and DOM manipulation |
